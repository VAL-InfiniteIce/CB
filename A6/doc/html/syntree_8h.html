Ausführliche Beschreibung

Enthält eine Datenstruktur für abstrakte Syntaxbäume.

Autor
Dorian Weber

Hier ist ein Beispiel für die Konstruktion eines Syntaxbaumes:

syntree_t ast;
syntree_nid node;
syntreeInit(&ast);
node = syntreeNodeTag(&ast, SYNTREE_TAG_Times, syntreeNodeInteger(&ast, 6));
syntreeNodeAppend(&ast, node, syntreeNodeInteger(&ast, 4));
node = syntreeNodePair(&ast, SYNTREE_TAG_Plus, syntreeNodeInteger(&ast, 2), node);
syntreePrint(&ast, node, stdout, 0);
syntreeRelease(&ast);

mit der Ausgabe

Plus {
Integer 2
Times {
Integer 6
Integer 4
}
}

Mithilfe der Knoten-IDs, die Ein- und Ausgabewerte der konstruierenden Funktionen sind, kann die Datenstruktur vom Blatt zur Wurzel konstruiert werden. Alle Knotentypen werden in einer X-Liste deklariert und die Literale (Blätter des abstrakten Syntaxbaumes) haben jeweils ein eigene Konstruktionsroutine, die den entsprechenden Wert als Parameter erhält. Knoten übernehmen im Falle von dynamisch allozierten Werten den Besitz und geben diese beim Abbau der Knoten selbständig frei.
